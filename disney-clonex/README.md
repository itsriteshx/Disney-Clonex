# 🎬 Disney+ Clone — React Web Application

A modern Disney+ inspired streaming platform clone built using **React.js**.  
This project demonstrates advanced React concepts including component-based architecture, routing, state management, and responsive UI design.

# 📌 Project Overview
This project is a front-end clone of the Disney+ streaming platform.  
It replicates the user interface and browsing experience of a real OTT platform.

Users can:

- Browse movies and TV shows
- Navigate between categories
- View dynamic movie rows
- Experience responsive layout
- Interact with modern UI components

This project focuses on **React fundamentals**, **UI/UX design**, and **component reusability**.
